# 解答例：3-1. `if` で step の実行条件を分ける

## 解答

`.github/workflows/3-1-if.yml` を以下の内容で新規に作成します。

```yaml
name: 3-1 if condition

on:
  workflow_dispatch:

jobs:
  job1:
    runs-on: ubuntu-latest
    steps:
      - name: Run job1
        run: echo "job1"

  job2:
    runs-on: ubuntu-latest
    needs: job1
    steps:
      - name: Conditional step
        if: needs.job1.result == 'success'
        run: echo "conditional step"
```

## 解説

- `if` に式を書くことで、その step の実行条件を指定できます。
- `needs.job1.result` で `job1` の実行結果を参照できます。
- `'success'` は `job1` が正常終了したことを表します。
- `job1` が失敗した場合、`conditional step` は実行されません。
